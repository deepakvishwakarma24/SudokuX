# SudokuX

SudokuX is an interactive Sudoku Solver built using the Recursive Backtracking Algorithm.

## Features

* Recursive Backtracking Solver
* Interactive 9×9 Sudoku Grid
* Real-time Input Validation
* Dark Modern UI
* Responsive Design
* Reset Functionality

## Tech Stack

* HTML5
* CSS3
* JavaScript
* Backtracking Algorithm
